Khode Account Theme is a modern, customizable Keycloak Account UI theme built with React, TypeScript, and Tailwind CSS.

## What is it?

A complete replacement for Keycloak's default account management interface that provides:

- Clean, intuitive user interface
- Modern React-based architecture
- Easy customization and branding
- Dark/light theme support
- Mobile-responsive design

## Key Features

- **Modern Stack**: React 18, TypeScript, Tailwind CSS
- **Fast Development**: Hot module replacement and live reload
- **Customizable**: Easy logo, color, and layout changes
- **Production Ready**: Secure and performant
- **Open Source**: MIT licensed

## Quick Setup

```bash
git clone https://github.com/khode-io/khode-account-theme.git
cd khode-account-theme
pnpm install
pnpm run dev
```

Then start Keycloak:
```bash
pnpm run start-keycloak
```

Visit http://localhost:8080 and go to "Manage Account" to see the custom theme.

## Use Cases

- **Enterprise**: Brand your account management interface
- **SaaS**: Provide better user experience
- **Developers**: Modern tooling and fast development
