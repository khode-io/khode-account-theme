Learn how to customize the logo, colors, and branding elements.

## Changing the Logo

### Method 1: Update Image URL

Edit `src/components/Header.tsx`:
```tsx
<img
  src="/your-logo.png"
  alt="Your Company"
  className="w-16 h-16 mr-3 object-contain"
/>
```

### Method 2: Add to Public Assets

1. Add your logo to `public/logo.png`
2. Update the Header component:
   ```tsx
   <img src="/logo.png" alt="Your Company" />
   ```

### Method 3: Theme-Aware Logo

Support different logos for light/dark themes:
```tsx
import { useTheme } from '../contexts/ThemeContext';

const { theme } = useTheme();
const logoSrc = theme === 'dark' ? '/logo-dark.png' : '/logo.png';

<img src={logoSrc} alt="Your Company" />
```

## Customizing Colors

### Primary Brand Colors

Edit `src/index.css`:
```css
:root {
  /* Your brand colors */
  --color-primary-500: 59 130 246;   /* Main brand color */
  --color-primary-600: 37 99 235;    /* Darker variant */
  
  /* Apply to theme */
  --color-accent-primary: var(--color-primary-600);
  --color-border-focus: var(--color-primary-500);
}
```

### Quick Color Change

Update key colors:
```css
:root {
  --color-accent-primary: 34 197 94;        /* Green example */
  --color-accent-primary-hover: 22 163 74;  /* Hover state */
}
```

## Dark Theme Colors

Customize dark theme:
```css
:root[data-theme="dark"] {
  --color-background: 13 17 23;
  --color-surface: 22 27 34;
  --color-text-primary: 201 209 217;
  --color-accent-primary: 88 166 255;
}
```

## Text Content

### Application Title

Edit `src/components/Header.tsx`:
```tsx
<h1 className="text-lg font-semibold text-text-primary mb-1">
  Your Company Account
</h1>
```

### Page Navigation

Update labels in `src/PageNav.tsx` and page components in `src/pages/`.

## Custom Fonts

Add web fonts in `src/index.css`:
```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap');

body {
  font-family: 'Inter', sans-serif;
}
```

## Environment Variables

Create `.env` for build-time customization:
```env
VITE_APP_TITLE="Your Company Account"
VITE_BRAND_COLOR="#3b82f6"
VITE_LOGO_URL="/your-logo.png"
```

Use in components:
```tsx
const appTitle = import.meta.env.VITE_APP_TITLE || "Account";
```

## Testing Changes
